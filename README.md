# fail2ban-blocklist
Block IP addresses provided by blocklist.de via fail2ban

## Install via Ansible
You can install this integration by:

* Cloning this git repository to the target host
* `cd fail2ban-blocklist/ansible`
* `ansible-playbook install.yml`
